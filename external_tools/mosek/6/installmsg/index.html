MOSEK installation notes

What is next

The MOSEK installation has updated the operating system environment variables. The update does NOT take effect until next login. Therefore please reboot or logout before trying to use MOSEK.

For important information about this release please see the release notes.

License installation

Instructions on installing a commercial MOSEK license can be found in the MOSEK installation manual.

Installing a trial license

To install a trial license, replace the file 

<INSTALL_DIR>\mosek\6\licenses\mosek.lic

with your trial license file send to you by email. <INSTALL_DIR> is the location of your MOSEK installation.

Setup of the MOSEK interfaces

For information on how to use MOSEK from your preferred API please consult the appropriate API manual available from. the MOSEK helpdesk.
