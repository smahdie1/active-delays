5.2. Additional reading

In this manual it is assumed that the reader is familiar with mathematics and in particular mathematical optimization. Some introduction to linear programming is found in books such as “Linear programming” by Chvátal [20] or “Computer Solution of Linear Programs” by Nazareth [11]. For more theoretical aspects see e.g. “Nonlinear programming: Theory and algorithms” by Bazaraa, Shetty, and Sherali [17]. Finally, the book “Model building in mathematical programming” by Williams [1] provides an excellent introduction to modeling issues in optimization.

Another useful resource is “Mathematical Programming Glossary” available at

http://glossary.computing.society.informs.org
